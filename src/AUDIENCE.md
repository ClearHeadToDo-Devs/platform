# Who is This For?
This repository is for those who are wanting to understand how all the pieces fit together WITHOUT diving into each product itself.

We will be coving high level guidance on what the platform is, its high-level objectives, and how to use the different products effectively to create a good experience


## Who is this NOT for?

This is not for those who are looking to contribute or those looking for in-depth technical documentation.

For a more technical overview of the platform, Please simply review the [platform repo README](https://github.com/ClearHeadToDo-Devs/platform/blob/main/README.md) which provides not only an index, but is linkable to the other products where more technical documentation is maintained.
